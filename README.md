# ダンロードフォルダ整理スクリプト

## 概要
このスクリプトは、`Downloads` フォルダ内のファイルを拡張子ごとに整理します。

## 使い方
1. Pythonをインストール（必要なら）
2. スクリプトを実行
      ```sh
   python organize_files.py
3. 整理後のフォルダ構成
    /Downloads
        ├── PDFs
        │   ├── document.pdf
        ├── Images
        │   ├── photo.jpg
        ├── Scripts
        │   ├── script.py
        ├── Texts
        │   ├── notes.txt
        ├── Music
        │   ├── song.mp3
        ├── Others
        │   ├── unknown_file.xyz
